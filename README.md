Project3: Валютный кошелек 

Консольное приложение для имитации работы валютного кошелька.

платформа для управления виртуальным портфелем фиатных и криптовалют. Поддерживает регистрацию пользователей, сделки (покупка/продажа), просмотр портфеля, получение курсов в реальном времени из внешних API (CoinGecko для крипты, ExchangeRate-API для фиата). Данные хранятся в JSON, логи ротируются, обновление курсов — автоматически или вручную через CLI.
После установки и запуска выполните следующие команды для работы с платформой:

register --username <имя> --password <пароль> — зарегистрировать пользователя (пароль ≥4 символа).
login --username <имя> --password <пароль> — войти в систему (сессия сохраняется).
logout — выйти из системы.
buy --currency <код> --amount <количество> — купить валюту (e.g., BTC 0.01).
sell --currency <код> --amount <количество> — продать валюту (проверка баланса).
show-portfolio [--base <валюта>] — показать портфель (таблица балансов, итого в USD/EUR).
update-rates [--source <coingecko/exchangerate/all>] — обновить курсы (все или по источнику).
show-rates [--currency <код>] [--top <число>] [--base <валюта>] — показать курсы (фильтры, топ крипты).
help — справка.
exit — выход (или Ctrl+C).

Пример:
Введите команду: register --username alice --password 1234
Пользователь 'alice' зарегистрирован (id=1). Войдите: login --username alice --password ****
Введите команду: login --username alice --password 1234
Вы вошли как 'alice'
Введите команду: buy --currency BTC --amount 0.01
Покупка выполнена: 0.0100 BTC по курсу 96066.00 USD/BTC
